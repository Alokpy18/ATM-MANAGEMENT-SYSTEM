# ATM-MANAGEMENT-SYSTEM
A simple menu-driven ATM MANAGEMENT SYSTEM Developed using python.

# FEATURES :
1. CHECK BALANCE : It is check the balance how much balance in your bank account.
 
2. DEPOSIT MONEY :  It is deposit a money in your bank account and tell the customer how much current balance in the customer bank account.
   
3. WITHDRAWAL MONEY : It is withdrawal monwy in your account and tell the customer how much current balance in the customer bank account.

4. PIN CHANGE : It is change a pin to firstly enter the old pin in the atm machine and second create a new pin and enter the pin in new pin block and third confirm the new pin in the atm and the pin are changed successfully.

5. EXIT PROGRAM : And the customer are click the exit button and come out the program.


# TECHNOLOGIES USED :

PYTHON 3 :
1. IF : IT IS USED FOR DECISION-MAKING AND CONTOLLING THE FLOW OF A PROGRAM.
2. ELSE : IT IS KEYWORD SERVES AS A FALLBACK OR A "CATCH-ALL" CONTROL STRUCTURE.
3. ELIF : TO LINK THE MULTIPLE CONDITION SEQUENTIALLY WITHIN A SINGLE CONDITION BLOCK. 
4. FLOAT : IT IS USED FOR WRITE A DECIMAL NUMBER IN PYTHON PROGRAM.
5. INT : IT IS USED FOR WRITE A INTEGER VALUE IN PYTHON PROGRAM. 
6. INPUT : IT IS USED TO TAKE DATA DIRECTLY FROM THE USER. WHEN THIS FUNCTION RUNS, THE PROGRAM POUSES AND WAITS FOR THE USER TO TYPE SOMETHING ON THEIR KEYBOARD AND PRESS ENTER.

# HOW TO RUN THE PROGRAM :

1. Download the project files.
2. open the project folder in VS code or any python ide.
3. run the main python file.
   '''bash
   python main.py
   '''
